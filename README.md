# Nebula

基于 Rust 的词库引擎 / 文本处理 DSL 解释器。读取 `.nr` / `.n` 脚本文件，通过触发词匹配执行文本处理逻辑，适用于聊天机器人、互动小说、自动化文本生成等场景。

## 快速开始

```bash
git clone https://github.com/your/nebula.git
cd nebula
cargo run
```

默认运行 `dic.nr` 文件。修改 `src/main.rs` 中的 `trigger` 变量切换触发词，或设为空字符串执行全词库。

## 文件结构

| 文件 | 说明 |
|---|---|
| `src/main.rs` | 入口，决定运行模式 |
| `src/parser.rs` | 解析 `.nr` 词库文件 |
| `src/interpreter.rs` | 核心状态机解释器 |
| `src/value.rs` | 变量存储、`%变量%` 替换、JSON 导航 |
| `src/count.rs` | `[...]` 数学表达式求值 |
| `src/analyzer.rs` | 赋值操作符检测 |
| `src/iftext.rs` | 条件表达式求值（支持短路） |
| `src/functions.rs` | 内置函数：`$回调$` `$跳行$` `$打印$` `$打印返回$` |

## 基本语法

### 词条

```
触发词
词条内容第一行
词条内容第二行
```

触发词支持正则表达式（如 `.*` 匹配所有输入），按文件顺序优先匹配。

### 变量与赋值

```nr
# 头部变量（全局）
name:World

# 普通赋值（执行 $...$ 和数学表达式）
name:Alice
score:85

# 纯文本赋值（不处理）
greeting::Hello World

# 数学运算
score+:5           # 自增
score-:3           # 自减
score*:2           # 乘法
score/:2           # 除法

# 引用变量
%name%             # 输出变量值

# 特殊变量
%空格%             # 空格
%换行%             # 换行
%时间戳%           # Unix 时间戳（秒）
%毫秒时间戳%       # Unix 时间戳（毫秒）
%1-100%            # 随机整数 [1, 100]
%!key%             # 布尔取反

# 数学表达式
result:[%score% * 2 + 10]
```

### 条件判断

支持 `==` `!=` `>=` `<=` `>` `<` `~=` `in`，逻辑操作符 `&&` `||`（支持短路求值）。

**行内语法：**

```nr
如果:%score%>=60
及格
否则
不及格
如果尾
```

**块语法：**

```nr
如果>%score%>=90
优秀
>否则如果:%score%>=60
及格
>否则
不及格
<如果
```

### 循环

```nr
循环>i=5
第%i%行
<循环

遍历>key,val=@jsonData
%key%: %val%
<遍历
```

### JSON 操作

```nr
data:{"name": "Alice", "age": 30}
@data[name]        # 导航读取 → "Alice"
data[age]:25       # 导航写入
```

### `$...$` 回调与内置函数

```nr
$say_hello %name%$           # 调用内部词条
$回调 pattern arg1 arg2$     # 正则可调
$跳行 [%行数%+2]$            # 跳转到绝对行号
$打印 content$               # 输出不返回
$打印返回 content$           # 输出并返回
```

### 控制指令

| 指令 | 说明 |
|---|---|
| `>终止` | 停止所有执行 |
| `>终止 msg` | 停止并输出 |
| `>跳过` | 跳过当前循环/遍历/判断 |
| `>终止循环` | 跳出循环 |
| `>终止遍历` | 跳出遍历 |

### 多行文本

```nr
text:"""
多行文本内容
支持插入 %变量%
"""

text:'''
单行原样文本
'''
```

### 内部词条与函数

```nr
[内部]say_hello
你好，%参数1%！

[函数]print_num
第%i%行
```

### 注释

```nr
// 单行注释
/* 多行
   注释 */
```
