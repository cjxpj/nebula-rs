# $读文件$ — 读取文件

基础函数 | `#引入=@文件`

<dl>
  <dt>签名</dt><dd><code>$读文件 [路径] [默认值]$</code></dd>
  <dt>参数</dt><dd>文件路径、可选的默认值</dd>
  <dt>返回值</dt><dd>文件全部内容；文件不存在或失败时返回默认值（省略时返回空字符串）</dd>
</dl>

读取文件的全部内容作为字符串返回。读取时自动将 `\r\n` 和 `\r` 换行符归一化为 `\n`。

```
$读文件 data/config.json 默认配置$
```

> 建议在关键文件操作前使用 `$存在文件$` 做前置检查。
