# $替换$ — 替换子串 <span class="badge">基础</span>

导入：`#引入=@字符串` | 标准库函数。

<dl>
  <dt>签名</dt><dd><code>$替换 [字符串] [旧] [新]$</code></dd>
  <dt>参数</dt><dd>源字符串、要替换的旧子串、新子串</dd>
  <dt>返回值</dt><dd>替换后的字符串</dd>
</dl>

将字符串中**所有**出现的旧子串替换为新子串（全局替换，非仅首个匹配）。

```
$替换 hello world o x$   → "hellx wxrld"
```

> 属于基础函数，无需导入即可使用。
